# ⚡ GETTING STARTED (30 Detik)

## 1️⃣ Buka PowerShell/Terminal

```powershell
# Navigate ke project
cd D:\up2github\DotNetCli\TextMatrix
```

## 2️⃣ Jalankan Aplikasi

```powershell
dotnet run
```

## 3️⃣ Gunakan Menu

```
╔═════════════════════════════╗
║ Matrix Animation Console    ║
╚═════════════════════════════╝

Select option (1-3):
  1. Animate Text (Quick)
  2. Settings
  3. Exit

→ Press 1
```

## 4️⃣ Masukkan Teks

```
Enter text to animate:
→ Type: RANGGA
→ Press Enter
```

## 5️⃣ Saksikan Animasi! 🎬

```
ｦ
  ｧ
    ｨ      R A N G G A
      ｩ
        ｪ
```

- Teks "RANGGA" di tengah dengan efek glow
- Karakter jatuh dari atas dengan gradient warna hijau
- Press ESC untuk stop lebih awal
- Press any key untuk kembali ke menu

## 6️⃣ Keluar

```
Select option (1-3):
  → Press 3
→ Goodbye!
```
