## 🎯 Quick Reference

### Command Cheat Sheet

```powershell
# Build & Run
dotnet build
dotnet run

# Build Release
dotnet build --configuration Release

# Publish Standalone
dotnet publish -c Release -r win-x64 --self-contained

# Clean
dotnet clean
```

### Menu Navigation

```
Main Menu:
  1 = Animate Text
  2 = Settings
  3 = Exit

Settings:
  1 = Duration (1000-15000 ms)
  2 = Frame Delay (20-100 ms)
  3 = Speed (1-5)
  4 = Trail Length (5-20)
  5 = Density (1-5)
  6 = Reset
  7 = Back
```

### Keyboard Shortcuts

- **ESC** = Stop animasi
- **Enter** = Confirm input
- **Any Key** = Continue dari hasil
